ShakerInstances

The physics backbone of SpringShaker. Implements a damped harmonic oscillator using the closed-form solution: 

S(t) = F(t) * cos(sqrt(T) * t) * e^(-D*t) * E(t)

Where:

  • F(t) — External noise force (Perlin rumble + jitter) scaled by Magnitude
  • cos(sqrt(T) * t) — Oscillation frequency driven by Tension
  • e^(-D*t) — Exponential decay driven by Damper (only active during fade-out)
  • E(t) — Cubic fade envelope CurrentTime^3

Not intended to be used directly — SpringShaker manages all instances internally.

Update​

</>
ShakerInstances:Update(
dx: number--

Delta time in seconds (typically from RenderStepped)

) → Vector3--

The current position offset to accumulate onto the camera

Steps the physics simulation forward by dx seconds. Internally this:

  1. Advances the cubic fade envelope (CurrentTime^3)
  2. Samples Perlin noise for rumble and jitter
  3. Applies the damped harmonic oscillator formula
  4. Clamps position to prevent explosive values
NOTE

Calling Update on an inactive shaker will return Vector3.zero and do no work.

CAUTION

Setting Magnitude above 10^5 risks Resonance — a real physics phenomenon where the forcing frequency matches the system's natural frequency, producing explosive and unregisterable values. SpringShaker will abort and warn if this is detected.

FadeOut​

</>
ShakerInstances:FadeOut(
Time: number--

Duration in seconds. Pass 0 for an immediate stop.

) → ()

Begins fading the shake out over the given duration. Sets SustainComplex to false, enabling the exponential decay envelope on the next Update. Resets RawTime to 0 so decay always starts from e^0 = 1.

FadeIn​

</>
ShakerInstances:FadeIn(
Time: number--

Duration in seconds. Pass 0 to start at full magnitude immediately.

) → ()

Begins fading the shake in over the given duration. Sets SustainComplex to true, holding decay at 1 until FadeOut is called.

GetFadeMagnitude​

</>
ShakerInstances:GetFadeMagnitude() → number

Returns the current fade multiplier in the range [0, 1]. Computed as CurrentTime^3 for a cubic ease curve — slow to start, fast to finish.

IsDead​

</>
ShakerInstances:IsDead() → boolean

Returns true if the shaker has fully decayed and is safe to recycle. Checks that SustainComplex is false, CurrentTime is at zero, and both Position and Velocity magnitudes are below the settlement threshold.
