SpringShaker

A physics-based camera shaker for Roblox built on a damped harmonic oscillator. Unlike noise-only shakers, SpringShaker simulates real mass and resistance — giving shakes a natural ring-down and weight.

SpringShaker.new(
__SpringShakeDefClass: Batch.__camShakePreset--

The configuration table for the shaker.

) → __SpringShakerClassDef--

Returns the shaker class object.

Creates a brand new Springy camera shaker instance.

Usage:

local MySpringShaker = SpringShaker.new({
    Magnitude = 0.5,
    Roughness = 0.3,
    FadeInTime = 0.2,
    FadeOutTime = 0.6,
    Tension = 120,
    Damping = 8,
    Velocity = Vector3.new(0, 0, 0),
    RotationInfluence = Vector3.new(1, 1, 1),
})
SpringShaker:ShakeOnce(MySpringShaker, 2) -- Shakes for apparent 2 seconds.

SpringShaker:Halt(
SpringDef: __SpringShakerClassDef--

The instance to halt.

) → ()

Forces a specific shaker instance to stop rendering immediately.

This does not use FadeOut method unlike :HaltDurationWise, so it will just stop the shaker without any fade effect. Used in benchmarking or emergency stops.

SpringShaker:ShakeIndividual(
SpringDef: __SpringShakerClassDef,--

The preset configuration.

Duration: number?--

How long before fadeout triggers.

) → ()

Plays a shake once, but creates a fully independent instance per call. Safe for rapid fire — each call lives and dies on its own without stomping previous shakes. Useful for gun recoil, footsteps, or any shake that may be triggered multiple times in quick succession.

Usage:

local SpringShaker = require(path.to.SpringShaker)
SpringShaker:ShakeIndividual(SpringShaker:GetPreset("M2Recoil"), 0.3)
